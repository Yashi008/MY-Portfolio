SynthAIsum

A Flask web application that performs text summarization and evaluation on user-provided text or uploaded documents. The application uses various natural language processing libraries such as NLTK, Rouge, and SacreBleu to extract sentences, calculate scores, and generate summaries. The user can input text or upload a document, choose the number of sentences for the summary, and view the summary along with the evaluation scores. The application also generates a graph to visualize the Rouge scores for different score types.

AyurPharmaGuide

This is a Flask web application that uses machine learning to predict diseases based on user-inputted symptoms. The application uses KNeighborsClassifier for classification. It also provides a classification report and suggests medicines based on the predicted disease. The application also includes a preprocessing function to clean the input data and a function to retrieve symptom descriptions from a dataframe. Additionally, the application uses a Naive Bayes classifier to suggest medicines based on the user's input symptoms.
